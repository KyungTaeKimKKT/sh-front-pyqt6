2. BASE SW 언어 : Python

3.  구축 개요
    -. 서버 :  
       1. DRF(  with uwsgi + nginx + redis 및 swagger 적용 ) + Postgresql DB
       2. WebSocket ( DJANGO CHANNEL + nginx + redis )
       3. WebServer ( React , Nginx )
       4. Dockerizse : DRF, WebSocket, WebServer  및 DOCKER Image push 하면 자동update됨
       5. Postgresql DB는 SERVER에 설치함.
      
        6. DRF 보안 : JWT 적용 및 각 Module 마다 permission 관리
        7. DRF 성능 보완을 위한 
            -. redis cache 적용
            -. client 요청에 따른 dynamic pagination 적용

   -. 클라이언트 ( PC WINDOW OS):
      1. 기본 : Native App으로 PyQt6을 이용한 GUI  프로그램 
      2. 추가 : webServer 접속 
      
    -. 추가 클라이언트 : mobile 장치에 native app ( Android, iOS )
        1. 예상 : React Native 또는 python kivy 
        2. 어떤것이 좋을 까 의견 주세요.

  4. 현재까지 완료된 사항 : 
      1) 서버 :  완료 ( module 단위로 update 하고 있음 )
      2) 클라이언트 ( pc) : 
          -. 자동 UPDATE : APP을 관리자가 등록하면 자동으로 강제적으로 UPDATE됨
